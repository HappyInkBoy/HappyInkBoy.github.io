This is the LAC instruction sheet. Here you can learn more about LAC's features and also find neat resources for further reading. Above is the linear algebra calculator with no inputs. To use it, adjust the number of columns and rows of a matrix using the plus and minus buttons, then input the numbers by clicking on the numbers and typing in the desired number. Then you can enter the letter of the matrix into the expression box and input a command to perform relevant calculations. To input a command, you can type it in or click on it. Then, click enter to see the result of the calculation in the ANS matrix. The table below will explain the result of each command as well as how to use them.
